# Fraud Detection Pipeline — Weeks 8-10

This repository contains the fraud detection ML pipeline for the Bread Financial Academy.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Structure

- `data/` — Transaction data
- `notebooks/` — Reference notebook from weeks 5-7
- `src/` — Production Python modules (you build these!)
- `tests/` — Test suite (Week 9)

## Dataset

`data/transactions_sample.csv` — 500 synthetic financial transactions with fraud labels (~5% fraud rate).
